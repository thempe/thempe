Thematic Map Types

As it is represented in the Figure 1 beside, we can see different types of thematic map out there. The type of data we are working on will influence the way we visualise it on thematic maps. For example, categorical data, like types of vegetation often leads to maps with different colours or patterns for each category.

The specific data type guides us in selecting the best thematic map style to effectively convey the information at hand.

Here are thematic map types we introduce on THEMPE:

Figure 2. Choropleth Map

1

Choropleth Map

As you can see in Figure 2 beside, this is the example of Choropleth Map to represent population density 2022.

➡️ it depicts quantitative data
➡️ it represents quantitative or enumerated data

It relies on colour value as visual variable to show the gradation. Also, it employs colour hue and saturation for advanced symbolization.

2

Proportional Symbol Map

From Figure 3, we can see the example of Proportional Symbol Map for mapping the oil consumption

➡️ it depicts quantitative data
➡️ it represents individual or enumerated data by mapping the attribute value as a centroid point of the polygon

Possible visual variable for this map is size

Figure 3. Proportional Symbol Map

Figure 4. Flow Map

3

Flow Map

In the figure 4, we can see how we visualise the data to a flow map

➡️ it depicts nominal and/or quantitative data
➡️ it represents the direction and/or magnitude of a certain occured phenomenon along linear objects or between locations

The most special thing about this map is that it can visualise both qualitative and quantitative data.

4

Nominal Map

From Figure 5, we can see the example of Nominal Map

➡️ it depicts nominal or categorical data
➡️ it represents individual or enumerated data

Possible visual variables for this map are colour hue and shape

Figure 5. Nominal Map

Figure 6. Dasymetric Map

5

Dasymetric Map

In the figure 6, we can see how we visualise the data to a dasymetric map

➡️ it depicts quantitative data
➡️ it represents the quantitative or enumerated data

The most special thing about this map is that it employs ancillary data to establish more meaningful boundaries for enumeration units, thereby enhancing the depiction of the geographic distribution of the phenomenon being mapped.

6

Bivariate Map

From Figure 7, we can see the example of Bivariate Map

➡️ it depicts two data attributes in one thematic map
➡️ it represents the legend as X and Y axis

A powerful map for the visual interpretation of spatial patterns of the geospatial data

Figure 7. Bivariate Map

Figure 8. Cartogram

7

Cartogram

In the figure 8, we can see how we visualise the data to cartogram

➡️ it depicts quantitative differences
➡️ it scales each geographical region according to its attribute value, and employs size as a visual element to represent the data

This map distorts the size, shape, and/or topological relationships of the enumeration units it visualises.

8

Diagram Map

From Figure 9, we can see the example of Diagram Map

➡️ it uses location as visual variable on map, and other variables (e.g. size) can be added to show qualitative and quantitative data.

By having diagrams on the map and visualise it on the centroid of enumeration unit, the non-geographic attributes and temporal pattern can be seen on it.

Figure 9. Diagram Map

Figure 10. Isoline Map

9

Isoline Map

In the figure 10, we can see the example of isoline map.

Thematic representation on a map created by drawing lines through interpolation, connecting points with the same attribute value.

These lines can be referred to as either isometric lines or isoplethic lines.

10

Dot Density Map

From Figure 11, we can see the example of Dot Density Map

➡️ it represents dots within a geographic region based on the value they represent, aiming to maintain the distribution and density variations of a specific phenomenon.

This map employs the visual variable called "numerousness" which blends arrangement and size to convey the impression of either sparseness or concentrated clustering.

Figure 11. Dot Density Map
