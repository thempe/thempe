What is Thematic Mapping?

Maybe you have tried to search "thematic maps" just like in Figure 1, right?

As it is shown in Figure 2, Thematic Cartography is one of the domains which is focused on observed or extracted attributes. Attributes and techniques are predominant in Thematic Cartography. Furthermore, the product of Thematic Cartography is the Thematic Map.

We can conclude that Thematic Mapping is a process of creating a thematic map, where we need to look into thematic cartography theory and apply it into the process.

Figure 3. Proportional Symbol Map - Total GDP

What is the goal of thematic mapping?

Thematic mapping is a cartographic process of making a thematic map. The thematic map itself is a map which can represent the various geographic phenomena(s), the location, and the information of its attribute at the same time. The goal of this map is to be able to show the audience about statistical information which usually can not be understood directly from tables or graphs. You can see Figure 3 as a thematic map example.

Good Map Examples

A good map always follows this quality below ⬇️

- shows an obvious idea which tells the purpose of the map
- visualizes with a good symbolization which is easy to read
- uses the data from a good source
- represents clear contents so it is not too crowded
- gives a legend which is easy to interpret

Where can I create a map?

You can use mapping software or graphic designing software applications. If you want to see what software you can use, you can open the page in Figure 5 beside.

The thematic map could be born from anywhere, but you should make sure that your thematic map follows the rules. Why? Because there is already a guideline of how to make a good map. This guideline exists to make sure that the map can be a tool for effective communication between map maker and map reader.
